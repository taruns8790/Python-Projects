Web Development using Flask and Python:

In the project, a techinical blog is created where user can add blog posts.
Also, a client can leave his contact details through the form and these details along with posts details are stored in SQL.
 


LIBRARIES IMPORTED inthe project:

from flask import Flask, render_template,request,session,redirect
from flask_sqlalchemy import SQLAlchemy


   1.The app can be run Run on browser by using http://127.0.0.1:5000/
   2. User can go through the blog to check various details such as contact and 
      about us details.
   3. We can also add new posts to the blogs.
   4. All the details of posts and client details is stoted in SQL.
